Parse API calls
==================


The API calls in the following sections are used to manage publishers, topics, subscriptions, and security for ingestion of OSIsoft Message Format (OMF) messages into OSIsoft Cloud Services (OCS). See `Introduction to Parse <https://qi-docs-rst.readthedocs.org/en/latest/parse_intro.html>`__ for more information.
